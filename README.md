# Projeto ETL com Python

Este projeto implementa um pipeline ETL (Extract, Transform, Load) utilizando Python.

## 🔹 Etapas

- Extract: leitura de dados de um arquivo CSV
- Transform: geração de mensagens personalizadas para cada usuário
- Load: salvamento dos dados em um novo arquivo CSV

## 🔹 Tecnologias

- Python
- Pandas
- OpenAI (para geração de mensagens - opcional)

## 🔹 Estrutura do Projeto

- Projeto_ETL_em_Python_e_IA.ipynb → código principal
- usuarios.csv → dados de entrada
- usuarios_final.csv → dados transformados

## 🔹 Objetivo

Praticar conceitos de engenharia de dados e pipeline ETL.
