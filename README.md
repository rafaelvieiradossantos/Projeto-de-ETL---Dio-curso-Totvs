# 🚀 Projeto ETL com Python e OpenAI

Este projeto demonstra a construção de um pipeline de dados utilizando o processo ETL (Extract, Transform, Load), aplicando conceitos fundamentais de Engenharia de Dados com Python.

## 📌 Objetivo

Desenvolver um fluxo completo de dados, desde a extração até o carregamento, simulando um cenário real de tratamento e enriquecimento de dados.

---

## 🔄 Pipeline ETL

### 🔹 Extract (Extração)
Leitura de dados a partir de um arquivo CSV contendo informações de usuários.

### 🔹 Transform (Transformação)
- Limpeza e padronização dos dados  
- Geração de mensagens personalizadas para cada usuário  
- (Opcional) Integração com OpenAI para criação automatizada de conteúdo  

### 🔹 Load (Carregamento)
Persistência dos dados transformados em um novo arquivo CSV estruturado.

---

## 🛠️ Tecnologias Utilizadas

- OpenAI (opcional para geração de mensagens)
